# Assistant Virtuel (Chatbot PHP)

Ce projet est un chatbot interactif développé en PHP et JavaScript.

## 🛠️ Technologies utilisées
* Backend : PHP 8
* Frontend : HTML5, CSS3, Vanilla JS
* Base de données : MySQL (si tu en as utilisé une)

## 📁 Architecture du projet
* `index.php` : L'interface principale du chat.
* `bot_logic.php` : Le script qui analyse les messages et génère les réponses.
* `/assets/` : Dossier contenant le style CSS et les scripts JS.

## 🚀 Comment tester le projet en local
1. Cloner le dépôt.
2. Placer le dossier dans le répertoire `www` (WAMP) ou `htdocs` (XAMPP).
3. Lancer un serveur local et accéder à `localhost/chatbot-php/index.php`.
